\begin{abstract}
This appendix presents a correctness proof of the simple version of the transformation for \SLCO that replaces a synchronous channel with an asynchronous channel.
We use the operational semantics of \SLCO presented in Appendix~\ref{ap:sos-slco} and the formal description of the transformation presented in Section~\ref{sec:transformations-slco:simple}.
A more concise description of this proof is given in Section~\ref{subsec:reusable-correct-transformations:correctness_of_simple}.
\end{abstract}
